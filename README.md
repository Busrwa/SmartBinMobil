# 🗑️ Smart Bin – Mobile App

React Native mobile client for Smart Bin, a full-stack IoT waste management
system (BSc Software Engineering Graduation Project, 2026).
Monitors bin fill levels and lid status in real time via Firebase.

## ⚙️ Tech Stack
- React Native + Expo
- Firebase Realtime Database (push listeners)

## 💡 Features
- Live bin fill level and lid status monitoring
- Firebase push listeners — no polling, ~150ms latency
- Cross-platform: iOS & Android

## 🔗 Related
- Web Dashboard: [SmartBinWebsite](https://github.com/Busrwa/SmartBinWebsite)

## 🚀 Getting Started
```bash
git clone https://github.com/Busrwa/SmartBinMobil.git
cd SmartBinMobil
npm install
npx expo start
```
